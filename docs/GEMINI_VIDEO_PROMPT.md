# Gemini video prompt — AeroCraft

Use this verbatim or as a storyboard for a 60–90 second product video.

**Subject:** AeroCraft, a CSS shortcut system from ForgeStack: one class expresses a full layout or typography intent; PostCSS emits plain CSS. Official site: https://aerocraftjs.com

**Visual style:** Clean developer-tool aesthetic. Light and dark UI (slate backgrounds, blue primary, indigo accent). Monospace for class names; subtle grid or code-editor chrome. No competitor logos or trademarked training data names on screen; say “utility CSS” or “multi-class stacks” if comparing patterns.

**Scene 1 — Hook (0–12s):** Split screen. Left: a long chain of utility-class strings for “centered row + gap + clamped text.” Right: a single AeroCraft class that produces the same computed styles. Title card: “One intent. One class.”

**Scene 2 — What it is (12–28s):** Full-width IDE or browser. Show `npm i @forgedevstack/aerocraft`, a tiny `postcss.config` with `aerocraftPlugin`, and a CSS file with `@aerocraft`. Cut to generated CSS in DevTools proving real declarations (flex, line-clamp, etc.).

**Scene 3 — Portal (28–45s):** Screen recording of aerocraftjs.com: docs sidebar, one reference page, Studio or Playground toggling prefix and groups, live preview updating.

**Scene 4 — Extras (45–60s):** Briefly show `theme` tokens (colors → utilities), optional `componentRecipes` (circle button / rounded input), and ForgeStack context (Bear, Compass, Rail at railjs.com).

**Scene 5 — CTA (60–75s):** URL on screen: aerocraftjs.com. Tagline: “Shortcut utilities. Real CSS. Ship anywhere.”

**Voiceover tone:** Confident, concise, no hype adjectives. Emphasize predictability, zero runtime, and framework-agnostic output.

**Music:** Neutral electronic bed, low under voice.

**Avoid:** Claiming file-size superiority without numbers; naming specific competitor products.
